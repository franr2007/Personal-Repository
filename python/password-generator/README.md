# Password Generator

Generador de contraseñas aleatorias por consola. El usuario elige la longitud y qué tipos de caracteres incluir.

## Como ejecutarlo

Requiere Python 3.12 o superior. No necesita dependencias externas.

python main.py

## Como funciona

Al elegir la opción 1 el programa pide:

- Longitud de la contraseña
- Si incluir letras (a-z, A-Z)
- Si incluir números (0-9)
- Si incluir símbolos (!@#$...)

Y genera una contraseña aleatoria con los caracteres seleccionados.
